# 👋 欢迎来到 AYue-cn 的小工具仓库！

嗨～ 我是 AYue-cn！这个仓库就像我的「数字工具箱」—— 里面全是我平时摸鱼间隙、遇到小麻烦时捣鼓的小工具，没什么高大上的功能，但胜在「解决自己的懒癌需求」很实用～

## 📦 仓库是啥？
简单说：**日常小需求的代码化解决方案**  
不是什么开源大作，就是自己踩了“重复劳动”的坑后，随手写的脚本/工具，比如：
- 批量处理文件（重命名、格式转换）
- 自动生成某类文档（比如周报模板、代码注释）
- 简化重复操作（比如自动登录某平台、数据批量录入）
总之，核心就是——**让懒癌患者少动手**！

## 🚀 怎么用？
1. 先把仓库克隆到本地：
   ```bash
   git clone https://github.com/AYue-cn/AYueProject.git  # 替换成你的仓库实际地址
   cd [你的仓库名]
   ```
2. 安装依赖（如果工具需要的话）：
   ```bash
   # 比如Python工具可能需要装依赖
   pip install -r requirements.txt
   ```
3. 运行对应工具：
   ```bash
   # 以各类脚本工具为例，替换为具体工具文件名即可
   python tools/[你的工具文件名].py
   ```
4. 跟着工具里的提示操作就行～ （每个工具里都写了简单的使用说明，放心！）
